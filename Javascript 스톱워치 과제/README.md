### 구현 내용

- setInterval을 사용하여 시간 측정
- localStorage를 사용하여 기록 저장
  - 삭제 시 blind 처리하지 않고 영구적으로 DOM과 localStorage에서 삭제
- 추가 구현
  - start, stop, reset 버튼을 눌렀을 때 버튼이 눌리는 것처럼 보이는 효과 추가
  - Javascript OOP를 사용하여 함수 추상화
